# 项目级交互规范 (Project-Level Interaction Guidelines)

## 核心写作原则 (Core Writing Principles)
**重要**:每次回答前都要叫我杰哥
### 1. 学术文体 (Academic Style)
- **段落化 (Paragraph-based)**: 正文必须采用连贯的段落式写作。**严禁**使用 Markdown 列表 (Bullet Points) 来堆砌解释性内容。列表仅限于真正的清单（如公式变量说明、特征列表）。
- **逻辑连接 (Logical Flow)**: 使用连接词（如"首先"、"此外"、"值得注意的是"、"综上所述"）来构建流畅的逻辑链条。
- **专业术语 (Terminology)**: 始终使用准确的学术术语（如"光合电子传递链"、"次生代谢"、"零样本推理"），并在首次出现时给出英文全称。
- **被动语态优先 (Passive Voice)**: 学术写作中优先使用被动语态（如"本研究采用..."而非"我们采用..."），保持客观性。
- **避免口语化表达**: 禁止使用"很好"、"非常"、"比较"等模糊表述，改用"显著(significantly)"、"较高(relatively high)"等精确表达。

### 2. 引用规范 (Citation Standards)
- **格式**: 必须统一使用 `(Author, Year)` 或 `(Author et al., Year)` 格式。
- **禁止**: 严禁使用 `[Journal, Year](Link)`、`[1]` 或直接粘贴 URL。
- **权威性**: 优先引用近3-5年的高水平文献（Nature/Science子刊、RSE、JXB、Plant Physiology等）。
- **引用密度**: 绪论部分每段至少2-3个引用；方法部分需引用原始方法来源；结果讨论需与前人研究对比。
- **自引适度**: 避免过度自引，保持客观性。
- **经典文献**: 对于奠基性工作（如Strasser的JIP-test），即使年代较早也应引用原始文献。

#### 2.1 前人研究引述规范 (Citing Previous Work)
引述前人研究成果时，必须采用"**研究者+研究内容+具体贡献**"的标准格式，使读者清晰了解学术脉络。

**标准格式**：
```
[研究者] (Year) [动词] [研究对象/方法]，[具体技术细节或创新点]，[主要成果或结论]。
```

**示例**：
- Zhang et al. (2023) 开发了一种用于小麦耐热性评价的自动化表型系统，该系统集成了铝合金框架、三台工业相机和嵌入式控制单元，利用LabVIEW程序实现多传感器协同采集，可提取颜色、形态、纹理及光谱特征共128个表型参数。
- Guo et al. (2022) 研制了以激光雷达为核心的高通量作物表型平台，集成高分辨率RGB相机、热成像仪和高光谱成像仪，通过移动轨道扫描获取株高、叶长、叶宽、叶面积等三维形态参数，测量精度达毫米级。
- Hollmann et al. (2023) 提出了TabPFN表格数据基础模型，基于Transformer架构在大规模合成数据集上进行预训练，实现了小样本场景下的零样本推理，在多个基准数据集上超越传统梯度提升方法。

**动词选择指南**：
| 研究类型 | 推荐动词 |
|:---------|:---------|
| 方法/系统开发 | 开发(developed)、研制(designed)、构建(constructed)、提出(proposed) |
| 理论/模型创新 | 提出(proposed)、建立(established)、推导(derived)、阐明(elucidated) |
| 实验/验证研究 | 验证(validated)、证实(demonstrated)、揭示(revealed)、发现(discovered) |
| 综述/分析工作 | 综述(reviewed)、总结(summarized)、分析(analyzed)、比较(compared) |
| 应用/推广研究 | 应用(applied)、拓展(extended)、优化(optimized)、改进(improved) |

**信息完整性要求**：
- **技术细节**: 需说明关键技术参数（如传感器型号、算法名称、样本量）
- **创新亮点**: 需点明该研究的独特贡献或突破点
- **量化成果**: 尽可能包含定量结果（如准确率、相关系数、提升幅度）
- **研究对象**: 明确作物种类、胁迫类型或应用场景

**禁止的引述方式**：
- ❌ "有研究表明..."（无具体出处）
- ❌ "前人研究发现..."（过于笼统）
- ❌ "Zhang研究了这个问题"（缺乏具体内容）
- ❌ "相关研究较多"（无实质信息）

**推荐的引述方式**：
- ✅ "Zhang et al. (2024) 采用深度卷积神经网络对水稻叶片高光谱图像进行分类，在干旱胁迫识别任务中达到96.3%的准确率，较传统SVM方法提升12.7个百分点。"
- ✅ "Strasser et al. (2004) 系统阐述了JIP-test方法的理论基础，定义了包括PIabs、Fv/Fm在内的30余个荧光参数，为OJIP曲线的定量分析提供了标准化框架。"

### 3. 图表规范 (Figures & Tables)
- **占位符**: 在正文中提到图表时，必须插入具体的占位符，例如：`[在此处插入 Figure 4-1: 六种模型性能对比雷达图]`。
- **说明**: 每个图表占位符后，应附带简短的说明文字，描述图表预期展示的内容和趋势。
- **先文后图**: 图表必须在正文首次提及之后出现，不可先于引用。
- **独立可读**: 图表标题和图注必须足够详细，使读者无需阅读正文即可理解图表内容。
- **编号连续**: 图表按章节编号（如图4-1、表5-2），全文连续不中断。
- **单位标注**: 所有数值必须标注单位，坐标轴必须有标签和单位。

### 4. 内容标准 (Content Standards)
- **发表级深度 (Publication-level Depth)**: 不仅要描述"是什么"，更要解释"为什么"。例如，不仅说模型效果好，还要解释其背后的数学原理（如 Prior-Data Fitting）和生理机制（如 K-step）。
- **毕业论文级充实度 (Thesis-level Fullness)**: 篇幅要足够长，综述要详尽，实验细节要毫无保留。
- **可重复性 (Reproducibility)**: 方法部分必须详细到他人可完全重复实验，包括仪器型号、软件版本、参数设置。
- **统计严谨性**: 所有比较必须有统计检验支持，报告p值、置信区间或效应量。

### 5. 创新点凸显 (Highlighting Innovation)
- **TabPFN**: 强调其作为"基础模型"在小样本、零样本场景下的革命性优势。
- **多源融合**: 强调"静态荧光"与"动态荧光"在生理层面的互补机制（光合 vs 代谢）。
- **可解释性**: 强调 SHAP 交互分析在揭示生物学机制中的独特价值。

## 章节特定规范 (Chapter-Specific Guidelines)

### 6. 绪论撰写规范 (Chapter 1: Introduction)
- **漏斗结构**: 从宏观背景（全球粮食安全）→ 具体问题（水稻干旱）→ 技术缺口 → 本研究定位，逐层聚焦。
- **研究意义**: 分理论意义、方法创新、应用价值三层次阐述。
- **文献综述**: 按技术演进或主题分类组织，而非简单罗列；每段需有小结句点明与本研究的关联。
- **现有不足**: 明确指出2-4个具体不足，每个不足直接对应本研究的一个创新点。
- **研究内容**: 用"（1）...（2）...（3）..."形式清晰列出，与后续章节一一对应。
- **技术路线图**: 必须包含，用流程图形式展示研究框架。

### 7. 材料与方法规范 (Chapter 2-3: Materials & Methods)
- **实验设计**: 说明样本量确定依据（统计功效分析）、随机化方法、重复次数。
- **仪器设备**: 列出品牌、型号、产地（如"SPAD-502叶绿素仪，Konica Minolta，日本"）。
- **试剂耗材**: 注明纯度、厂家、货号。
- **数据采集**: 详述采集时间、频率、环境条件。
- **数据处理**: 说明预处理步骤（异常值剔除标准、标准化方法）、软件版本（如"Python 3.10, scikit-learn 1.2.0"）。
- **公式编号**: 重要公式需编号（如式4-1），便于后文引用。
- **参数表格**: 模型超参数以表格形式呈现，包含参数名、取值、选择依据。

### 8. 结果撰写规范 (Chapter 3-5: Results)
- **结果-讨论分离或融合**: 明确采用哪种结构，保持全文一致。
- **数据呈现顺序**: 按研究问题的逻辑顺序，而非实验进行的时间顺序。
- **定量描述**: 避免"明显升高"等模糊表述，改用"升高了32.5%（p < 0.01）"。
- **不确定性**: 报告均值±标准差或标准误，明确n值。
- **图表引用**: 每个图表在正文中必须被引用和解释，不可仅放图不解读。
- **负面结果**: 如实报告不符合预期的结果，并给出可能的解释。

### 9. 讨论撰写规范 (Discussion Section)
- **结构层次**: 主要发现总结 → 与前人研究比较 → 机制解释 → 局限性 → 未来方向。
- **批判性思维**: 不仅解释为什么结果符合预期，也要讨论意外发现。
- **机制推测**: 基于证据进行合理推测，使用"可能(may)"、"推测(presumably)"等限定词。
- **局限性**: 坦诚讨论样本量、方法、外推性等方面的局限。
- **与引言呼应**: 讨论末尾应回应引言中提出的科学问题。

### 10. 结论撰写规范 (Chapter 6: Conclusion)
- **简洁有力**: 每条结论1-2句话，直接陈述发现，不重复方法。
- **对应研究目标**: 结论条目与引言中的研究目标一一对应。
- **避免过度延伸**: 结论必须有数据支持，不可超出研究范围。
- **创新点重申**: 明确列出本研究的原创贡献。
- **展望具体化**: 未来研究方向要具体可操作，而非泛泛而谈。

## 数学公式规范 (Mathematical Notation)

### 11. 公式书写标准
- **LaTeX格式**: 所有公式使用LaTeX语法，行内公式用 `$...$`，独立公式用 `$$...$$`。
- **变量斜体**: 变量用斜体（如 $x$, $\phi$），函数和运算符用正体（如 $\sin$, $\log$）。
- **下标规范**: 描述性下标用正体（如 $F_{\text{max}}$），变量下标用斜体（如 $x_i$）。
- **单位处理**: 数值与单位之间空格，单位用正体（如 $5\ \text{cm}$）。
- **公式编号**: 重要公式右侧编号，格式为（章-序号），如（4-1）。
- **变量定义**: 公式后必须定义所有变量，格式为"其中，$x$ 为..."。

### 12. 统计符号规范
- **均值**: $\bar{x}$ 或 $\mu$
- **标准差**: $s$ 或 $\sigma$
- **相关系数**: $r$（Pearson）或 $\rho$（Spearman）
- **显著性**: $p < 0.05$（斜体p）
- **样本量**: $n$（斜体）
- **效应量**: Cohen's $d$, $\eta^2$, $R^2$

## 语言润色规范 (Language Polishing)

### 13. 禁用词汇表 (Banned Words/Phrases)
| 禁用表达 | 替换为 |
|:---------|:-------|
| 很/非常/特别 | 显著/极显著/高度 |
| 好/差 | 优异/较差/欠佳 |
| 大概/可能/也许 | 约/推测/或许（需有依据） |
| 做了/进行了 | 开展/实施/采用 |
| 得到/获得 | 获取/测定/计算 |
| 说明/表明 | 证实/表征/揭示 |
| 有关系 | 相关/关联/耦合 |
| 比较好 | 较优/更佳/性能提升 |

### 14. 推荐学术表达 (Recommended Academic Phrases)
- **引出研究**: "本研究旨在..."、"鉴于...，本章拟..."
- **描述方法**: "采用...方法"、"基于...原理"、"通过...实现"
- **呈现结果**: "结果表明..."、"如图X所示..."、"数据显示..."
- **比较分析**: "与...相比"、"优于/劣于..."、"一致/矛盾于..."
- **解释机制**: "这可能归因于..."、"其生理基础在于..."、"从机理上讲..."
- **承认局限**: "需要指出的是..."、"本研究的局限性在于..."
- **展望未来**: "未来研究可聚焦于..."、"有待进一步验证..."

### 15. 段落结构规范
- **主题句**: 每段首句为主题句，概括本段核心内容。
- **支撑句**: 中间句子提供证据、数据、引用支持主题句。
- **结论句**: 末句总结本段或过渡到下一段。
- **段落长度**: 每段150-300字为宜，避免过短（<100字）或过长（>400字）。
- **段间过渡**: 使用过渡句或过渡词连接段落，保持逻辑流畅。

## 格式规范 (Formatting Standards)

### 16. 标题层级
- **一级标题**: `# 第X章 章标题`（居中，黑体，三号）
- **二级标题**: `## X.X 节标题`（左对齐，黑体，小三）
- **三级标题**: `### X.X.X 小节标题`（左对齐，黑体，四号）
- **四级标题**: `**（1）要点标题**`（段首，黑体，小四）
- **层级限制**: 最多使用四级标题，避免过度细分。

### 17. 数字与单位
- **阿拉伯数字**: 计量数值一律用阿拉伯数字（如"13个品种"、"37个特征"）。
- **中文数字**: 固定搭配用中文（如"第一章"、"三种模态"、"两方面"）。
- **百分比**: 使用"%"符号，前面不空格（如"32.5%"）。
- **范围**: 使用"~"或"至"（如"0.85~0.95"或"0.85至0.95"）。
- **SI单位**: 优先使用国际单位制，非SI单位需换算或注明。

### 18. 缩写规范
- **首次全称**: 缩写首次出现时给出全称，格式为"全称（缩写，Abbreviation）"。
- **常用缩写**: NDVI、SHAP、PSII等常用缩写可在首次定义后直接使用。
- **避免过多缩写**: 一章内新缩写不超过10个，否则影响可读性。
- **缩写表**: 论文前置部分应包含缩写对照表。

## 高级学术表达规范 (Advanced Academic Writing Standards)

### 21. 论证逻辑结构 (Argumentation Logic)

#### 21.1 "问题-方案"递进模式
每个研究领域的综述应遵循"**提出问题→分析局限→引出方案**"的三段式递进结构：

**模板**：
```
[技术/方法A] 虽然能够 [优点]，但存在 [局限性]。为解决这一问题，[研究者] (Year) 提出了 [技术/方法B]，通过 [创新机制]，实现了 [改进效果]。
```

**示例**：
> 传统的田间产量比较法虽然结果直接，但需要完成作物的完整生育期，周期漫长且受环境波动干扰。为解决这一问题，Araus et al. (2002) 提出了基于生理性状的间接选择策略，通过整合叶片相对含水量、渗透调节能力等早期可测指标，将评价周期缩短至营养生长期。

#### 21.2 "现象-机理"解释模式
描述实验发现时，必须从"现象描述"过渡到"机理解释"：

**模板**：
```
[现象描述]。这一现象的生理基础在于：[机理解释]。具体而言，[分子/细胞层面的详细机制]。
```

**示例**：
> 干旱胁迫下，OJIP曲线在300 μs处出现特异性的K-step信号。这一现象的生理基础在于：重度干旱导致放氧复合体（OEC）的锰簇受损，电子供体侧的水裂解反应受阻。具体而言，OEC损伤使得P680⁺的再还原速率下降，导致荧光信号在O-J相出现异常隆起。

#### 21.3 "对比-优选"论证模式
比较多种方法/模型时，采用"各有优劣→本研究选择→理由充分"的结构：

**模板**：
```
[方法A] 具有 [优点A]，但 [局限A]；[方法B] 虽然 [优点B]，却 [局限B]。综合考虑 [评价标准]，本研究选择 [最终方案]，原因在于：[具体理由]。
```

### 22. 数据呈现的语言模式 (Data Presentation Patterns)

#### 22.1 定量结果的标准表述
所有定量结果必须包含：**数值 + 变化方向 + 比较基准 + 统计显著性**

**标准格式**：
```
[指标] [变化方向] 了 [数值]%（从 [基准值] 至 [目标值]，p < [显著性水平]）
```

**示例**：
- ✅ "干旱胁迫下叶面积下降了54.8%（从11.89 cm²降至5.17 cm²，p < 0.01）"
- ✅ "TabPFN的Spearman相关系数达到1.000，较CatBoost（ρ = 0.978）提升2.2个百分点"
- ❌ "叶面积明显下降"（缺乏定量信息）
- ❌ "TabPFN效果更好"（缺乏具体数值）

#### 22.2 趋势描述的动词梯度
根据变化幅度选择恰当的动词：

| 变化幅度 | 上升动词 | 下降动词 |
|:---------|:---------|:---------|
| <10% | 略有上升、小幅增加 | 略有下降、小幅减少 |
| 10-30% | 上升、增加、提高 | 下降、减少、降低 |
| 30-50% | 显著上升、大幅增加 | 显著下降、大幅减少 |
| >50% | 急剧上升、成倍增加 | 急剧下降、骤降 |
| 接近0或100% | 趋近于零、接近饱和 | 几乎消失、近乎完全丧失 |

#### 22.3 相关性描述的规范用语
根据相关系数绝对值选择描述词：

| |r| 或 |ρ| 范围 | 描述用语 |
|:-----------------|:---------|
| 0.00-0.19 | 极弱相关/几乎无相关 |
| 0.20-0.39 | 弱相关 |
| 0.40-0.59 | 中等相关 |
| 0.60-0.79 | 强相关/较强相关 |
| 0.80-0.99 | 极强相关/高度相关 |
| 1.00 | 完全相关/完美相关 |

### 23. 学术语气与立场表达 (Academic Tone & Stance)

#### 23.1 确定性程度的语言梯度
根据证据强度选择恰当的限定词：

| 确定性程度 | 表达方式 | 适用场景 |
|:-----------|:---------|:---------|
| 确定（有直接证据） | 证实、表明、揭示、阐明 | 本研究的直接实验结果 |
| 高度可能（有间接证据） | 强烈暗示、有力支持、充分说明 | 与前人研究一致的发现 |
| 可能（有理论支持） | 可能、推测、或许 | 基于机理的合理推断 |
| 初步（证据有限） | 初步表明、似乎暗示 | 样本量小或需进一步验证 |
| 假设（无直接证据） | 假设、推测、有待验证 | 理论推演或未来研究方向 |

#### 23.2 批判性评价的委婉表达
评价前人研究局限时，避免直接否定，采用"肯定-转折-建议"模式：

**模板**：
```
[前人研究] 在 [某方面] 取得了重要进展，然而，[局限性描述]。本研究拟在此基础上，[改进方案]。
```

**示例**：
- ✅ "Fischer & Maurer (1978) 提出的DSI指数在评价品种稳定性方面具有开创性意义，然而，该指数对环境互作效应的敏感性限制了其在多环境试验中的应用。本研究拟引入GGE双标图分析，以更全面地解析基因型×环境互作。"
- ❌ "Fischer的方法存在严重缺陷"（过于直接）
- ❌ "前人的研究都不够好"（过于笼统且不礼貌）

#### 23.3 创新点陈述的平衡表达
既要突出创新，又要避免过度自夸：

**推荐表达**：
- "据我们所知，本研究首次将...应用于..."
- "与现有方法相比，本研究的主要改进在于..."
- "本研究的独特之处在于..."
- "这一发现为...提供了新的证据/视角"

**避免表达**：
- ❌ "本研究具有重大意义"（自我评价）
- ❌ "这是一项突破性研究"（过度自夸）
- ❌ "前人从未研究过"（难以证实）

### 25. 段落衔接与过渡 (Paragraph Transitions)

#### 25.1 章节间过渡句
每章结尾应有承上启下的过渡段，预告下一章内容：

**模板**：
```
本章从 [本章主题] 角度验证了 [核心发现]。然而，[遗留问题] 尚未解决。为回答 [科学问题]，下一章将采用 [方法]，深入分析 [研究内容]。
```

**示例**：
> 本章通过消融实验验证了"多源融合优于单源"这一核心假设。然而，融合增益的内在机制尚不清楚——为什么三种模态的组合能够产生协同效应？为回答这一问题，下一章将采用SHAP交互值分析方法，深入挖掘跨模态特征间的非线性协同作用。

#### 25.2 段落间逻辑连接词库

| 逻辑关系 | 连接词/短语 |
|:---------|:------------|
| 递进 | 此外、进一步、更重要的是、不仅如此 |
| 转折 | 然而、但是、尽管如此、与此相反 |
| 因果 | 因此、由此可见、这表明、究其原因 |
| 并列 | 同时、与此同时、另一方面、类似地 |
| 举例 | 例如、以...为例、具体而言、典型案例是 |
| 总结 | 综上所述、总之、概言之、由此可见 |
| 时序 | 首先、其次、随后、最终、在此基础上 |
| 对比 | 与...相比、相较于、不同于、优于/劣于 |

#### 25.3 避免的过渡方式
- ❌ 无过渡的跳跃（段落间无逻辑连接）
- ❌ 机械重复（"下面介绍..."、"接下来讨论..."）
- ❌ 口语化过渡（"说到这个"、"顺便提一下"）

### 26. 图表引用的语言规范 (Figure/Table Citation Language)

#### 26.1 图表引用的完整句式
图表引用必须嵌入完整的解释性句子，不可孤立出现：

**标准格式**：
```
如 [图/表X-X] 所示，[主要发现/趋势描述]。其中，[具体数据点解读]。
```

**示例**：
- ✅ "如图4-1所示，六种模型的Spearman相关系数呈现明显的层级分布。其中，TabPFN以ρ = 1.000位居首位，显著优于第二名CatBoost（ρ = 0.978）。"
- ❌ "见图4-1。"（过于简略）
- ❌ "图4-1展示了结果。"（缺乏实质信息）

#### 26.2 图表解读的深度要求
图表解读必须包含三个层次：

1. **描述层**：客观陈述图表展示的内容
2. **分析层**：指出关键趋势、差异或模式
3. **解释层**：给出可能的原因或生物学意义

**示例**：
> 图5-3展示了37×37特征交互矩阵的热力图（描述层）。矩阵呈现明显的块状结构，OJIP-Static区块的交互强度显著高于其他区域（分析层）。这一模式表明，荧光动力学参数与静态荧光特征之间存在强烈的非线性耦合，可能源于两者共同反映了光合电子传递链在干旱胁迫下的功能重组（解释层）。

### 27. 方法描述的精确性要求 (Method Description Precision)

#### 27.1 实验条件的完整记录
所有可能影响结果重现性的条件必须明确记录：

**环境条件**：温度（昼/夜）、光照强度、光周期、相对湿度
**生物材料**：品种名称、来源、生育期、处理时的生长阶段
**仪器参数**：型号、测量波长、积分时间、采样频率
**软件版本**：编程语言版本、关键库版本（如Python 3.10, scikit-learn 1.2.0）

#### 27.2 算法描述的数学严谨性
算法描述应包含：

1. **数学定义**：核心公式及变量说明
2. **计算流程**：输入→处理步骤→输出
3. **参数设置**：超参数取值及选择依据
4. **实现细节**：随机种子、收敛标准、迭代次数

**示例**：
> TabPFN模型采用默认配置（N_ensemble_configurations=32），输入特征经Z-score标准化后直接送入预训练网络。模型通过In-Context Learning机制，在单次前向传播中完成从训练样本到测试样本的预测映射，无需梯度更新。所有实验设置随机种子为42以确保可重复性。

### 28. 讨论部分的深度要求 (Discussion Depth Requirements)

#### 28.1 结果解释的多层次分析
每个重要发现的讨论应涵盖三个层次：

1. **现象层**：本研究观察到什么
2. **机理层**：为什么会出现这一现象
3. **意义层**：这一发现有什么理论/应用价值

#### 28.2 与前人研究的对比模式
讨论中必须将本研究结果与前人研究进行对比：

**一致时**：
> 本研究发现OJIP_Vi与抗旱性呈强负相关（ρ = -0.72），这与Živčák et al. (2008) 在小麦中的发现一致，进一步证实了Vi作为跨物种通用胁迫指标的潜力。

**不一致时**：
> 与Feng et al. (2024) 报道的热红外特征主导地位不同，本研究发现OJIP荧光动力学贡献最大（51.34%）。这一差异可能源于：（1）作物种类不同（番茄 vs 水稻）；（2）胁迫类型不同（综合水分胁迫 vs 纯干旱胁迫）；（3）评价目标不同（胁迫检测 vs 抗性排名）。

#### 28.3 局限性讨论的诚实表达
局限性讨论应具体、诚实，并指出改进方向：

**模板**：
```
本研究存在以下局限性：[具体局限]。这可能导致 [潜在影响]。未来研究可通过 [改进方案] 加以解决。
```

**示例**：
> 本研究的样本量相对有限（n=117），可能影响模型在更大群体中的泛化能力。未来研究可通过扩大品种范围、增加胁迫梯度来验证模型的鲁棒性。此外，引入迁移学习策略可能有助于缓解小样本问题。

### 20. 多源光谱融合论文专项规范
- **模态命名**: 统一使用"Multi（多光谱反射）"、"Static（静态荧光）"、"OJIP（荧光动力学）"。
- **特征数量**: 全文统一表述为"37个特征"（19 Multi + 10 Static + 8 OJIP）。

## AI味检查规范 (AI-Generated Text Detection & Correction)

### 29. 禁止的AI写作模式

#### 29.1 先结论后论述（最典型的AI味）
AI倾向于先给出结论性判断，再展开论述。学术论文应先呈现数据/现象，再给出结论。

**禁止模式**：
```
❌ **水分相关指数的品种差异尤为显著**。NDWI仅下降22.2%...
❌ **光化学反射指数（PRI）对品种抗旱性的区分能力最强**。PRI反映...
❌ 抗旱型品种1252**表现出"主动防御激活"模式**：蓝色荧光...
```

**正确模式**：
```
✅ 在水分相关指数方面，NDWI直接反映叶片含水量状态。抗旱型品种1252的NDWI从0.18降至0.14，降幅为22.2%；...三类品种间的差异达2.9倍，表明抗旱品种具有更强的水分保持能力。
✅ 抗旱型品种1252的蓝色荧光BF(F440)升高幅度最大（+47.4%）...清晰地反映了"强防御+弱损伤"的协同模式。
```

#### 29.2 加粗小标题分隔段落
AI喜欢用加粗文字作为段落内小标题，破坏段落连贯性。

**禁止模式**：
```
❌ **形态学评价指标**是植物适应干旱最直观的表现。
❌ **荧光比值的品种差异深入解析**
❌ **不足一：单模态局限性**。现有研究多关注...
```

**正确模式**：
```
✅ 从形态学角度，根系构型是植物适应干旱的核心性状。
✅ 在六个荧光比值中，涉及蓝色荧光（F440）的比值呈现出最显著的品种间差异。
✅ 现有研究多关注单一的光谱或荧光模态，缺乏对OJIP光合动力学与多光谱反射、静态荧光的系统性整合研究。
```

#### 29.3 三段式法则（AI的标志性结构）
AI喜欢将内容机械地分为三点，使用编号列表呈现。

**禁止模式**：
```
❌ **不足一：...**
   **不足二：...**
   **不足三：...**
   针对上述三点不足，本研究提出相应的解决方案：（1）...（2）...（3）...
```

**正确模式**：
```
✅ 现有研究多关注单一的光谱或荧光模态...与此同时，在样本量有限的科研场景下...此外，多数研究止步于"黑箱预测"...
   针对上述不足，本研究拟从三个方面开展工作。在数据层面，...在模型层面，...在机制层面，...
```

#### 29.4 夸大性表述
AI倾向于使用绝对化、难以验证的表述。

| 禁用表达 | 替换为 |
|:---------|:-------|
| 率先/首次/最新 | 尝试/探索/近期 |
| 致力于/旨在 | 探索/拟/尝试 |
| 革命性/突破性 | 创新性/改进的 |
| 显著优于/远超 | 优于/高于（需有统计检验支持） |
| 完美/最优 | 较优/表现最佳（需限定条件） |

#### 29.5 过度使用"然而"
AI过度依赖"然而"进行转折，导致文风生硬。

**使用频率限制**：每500字最多1个"然而"

**替代表达**：
- 但/不过（轻度转折）
- 与此不同的是（对比转折）
- 需要指出的是（补充说明）
- 直接省略转折词，用句意自然过渡

### 30. AI味自查清单

撰写或修改文本后，逐项检查：

- [ ] 段落首句是否为结论性判断？（应改为现象描述或主题引入）
- [ ] 是否使用加粗文字作为段落内小标题？（应改为自然过渡句）
- [ ] 是否出现"**不足一/二/三**"或"（1）...（2）...（3）..."的机械结构？

- [ ] "然而"的使用频率是否过高？（每500字超过1个需修改）
- [ ] 引用是否放在结论之后？（应先引用建立理论基础，再呈现本研究数据）
- [ ] 数据呈现是否遵循"具体数值→变化趋势→结论解读"的顺序？

### 31. 段落结构的正确模式

#### 31.1 数据呈现段落
```
[指标介绍/测量方法] → [具体数值呈现] → [品种/处理间对比] → [差异倍数/统计显著性] → [生理机制解读] → [结论句]
```

**示例**：
> PRI（光化学反射指数）反映叶黄素循环的脱环氧化状态，是光合机构光能利用效率的瞬时指标。Gamon et al. (1992) 首次建立了PRI与光合利用效率（LUE）的定量关系。本研究中，抗旱型品种1252的PRI从-0.012降至-0.028，变化相对温和；敏感型品种1235则从-0.018骤降至-0.095，降幅达427.8%，表明其叶黄素循环的光保护机制已被严重激活。这一结果证实了PRI在区分品种抗旱性方面的高敏感性。

#### 31.2 方法比较段落
```
[方法A介绍+优点] → [方法A局限] → [方法B介绍+改进点] → [本研究选择+理由]
```

#### 31.3 综述过渡段落
```
[前人研究贡献] → [存在的问题/空白] → [与本研究的关联] → [本研究的定位]
```
