## 2. 处理响应的物理规律

### 2.1 干旱胁迫 (CK1 → D1) 的光谱变化

| 物理规律 | 机制 | 文献来源 |
|---------|------|---------|
| **R810↑ (水分减少)** | 水分子吸收NIR，水分减少→吸收↓→反射↑ | MDPI, NIH |
| R660变化不定 | 取决于叶绿素降解程度 | - |
| Fv/Fm↓ | 干旱损伤PSII反应中心 | 经典荧光理论 |

**关键引用**:
> "Under drought stress, near-infrared (NIR) reflectance in the 780-900 nm range is observed to **increase** as leaf water content decreases."
> — [MDPI Remote Sensing](https://mdpi.com)

> "随着叶片含水量的降低（水分胁迫），近红外区域（810nm等）的光谱反射率会**升高**。"
> — [plant-ecology.com](http://plant-ecology.com)

### 2.2 复水恢复 (D1 → RD2) 的光谱变化

| 物理规律 | 机制 | 数据验证 |
|---------|------|---------|
| **R810↓ (水分恢复)** | 水分恢复→NIR吸收↑→反射↓ | 77%品种符合 |
| **NDVI↓** | SPAD下降（生理重构期） | 85%品种符合 |
| Fv/Fm 持续下降 | 恢复期生理重构 | 77%品种符合 |

**注意**: 复水后NDVI下降是合理的，因为Phase 5报告显示SPAD的LC_recovery = 0.848 < 1。

## 7. 物理解释的完整故事

### 故事线

```
1. 我们有13个水稻品种，想评估它们的抗旱能力

2. 通过传统生理测量（株高、叶面积、SPAD等），用PCA-隶属函数法
   计算出综合抗旱指数 D_conv（Phase 5）

3. 同时，我们用光谱仪测量这些品种的光谱响应

4. 物理规律告诉我们：
   - 水分减少 → 近红外反射增加（水分子吸收NIR）
   - 叶绿素降解 → NDVI下降
   - 光合损伤 → Fv/Fm下降

5. 抗旱品种（高D_conv）应该：
   - 干旱时：水分保持好 → R810变化小
   - 复水时：恢复快 → 指标回升

6. 敏感品种（低D_conv）应该：
   - 干旱时：严重失水 → 光谱剧烈变化
   - 复水时：不可逆损伤 → 难以恢复

7. 因此，光谱特征可以预测D_conv，因为两者都反映了
   植物应对干旱胁迫的能力
```
